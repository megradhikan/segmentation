# assets/

Place demo media here. Files in this directory are **not tracked by git** (see root `.gitignore`).

| File | Description |
|---|---|
| `demo.gif` | Screen-capture of a full segmentation flow — used in the root README |
| `screenshot-*.png` | Static screenshots for the Results table in the README |

## Recording a demo GIF

Any screen-recorder that exports GIF works. Recommended tools:

- **macOS**: [Kap](https://getkap.co/) — free, exports GIF directly
- **Windows**: [ScreenToGif](https://www.screentogif.com/) — free and open-source
- **Linux / cross-platform**: `ffmpeg` + `gifsicle`

Keep the GIF under **5 MB** so GitHub renders it inline without prompting a download.
